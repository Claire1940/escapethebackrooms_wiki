<Section icon={Target} title="Locating the Color-Coded Cars">
The key to deciphering the **Escape the Backrooms elevator code** lies in finding a specific room containing four colored cars. This room is typically accessed through a distinct red door located within the initial area of Level 1. The layout of Level 1 is that of a massive concrete parking garage, filled with office rooms and scattered items like Almond Water and Flashlights.

Upon finding the red door, enter the room and observe the cars. Their colors and the count of each color will form the code. For example, if you find two red cars, one blue car, and one green car, the code will involve pressing the red button twice, then the blue button once, and the green button once. The exact order of colors might vary, but the number of times each color is pressed corresponds directly to the number of cars of that color.

Here's a breakdown of how the code works:

| Color Found | Number of Cars | Elevator Input |
|:------------|:---------------|:---------------|
| **Red**     | 2              | Press Red 2x   |
| **Blue**    | 1              | Press Blue 1x  |
| **Green**   | 1              | Press Green 1x |
| **Yellow**  | 0              | Press Yellow 0x|

<Callout type="warning" title="Randomized Code Alert">
The elevator code is **randomized for each game session**. Do not rely on codes found in previous runs or from other players. You must find the cars yourself every time.

<Section icon={Shield} title="Entering the Elevator Code: Step-by-Step">
Once you have identified the colors and their counts from the car room, head back to the elevator keypad. The keypad features buttons corresponding to the car colors. Input the code by pressing the colored buttons the correct number of times. The order in which you press the colors might not be strict in some versions, but matching the total count for each color is paramount.
