### Using Cheat Engine

One common method to increase the player count is by using a tool called Cheat Engine. This program allows users to modify memory values in running applications, effectively tricking the game into accepting more players.

<Steps>
  <Step num="1" title="Download Cheat Engine">
    Navigate to `cheatengine.org` and download the latest version of Cheat Engine. Install it on your computer following the on-screen instructions.
  </Step>
  <Step num="2" title="Launch Escape the Backrooms">
    Start the game and navigate to the multiplayer lobby creation screen. Set the initial player count to **4**.
  </Step>
  <Step num="3" title="Attach Cheat Engine to Game">
    Open Cheat Engine. Click the "Select a process to open" icon (usually a computer monitor with a magnifying glass) and select the "Escape the Backrooms" process.
  </Step>
  <Step num="4" title="Scan and Modify Value">
    In Cheat Engine, enter `4` in the "Value" field and click "First Scan". Then, change the in-game player count to `2` and perform a "Next Scan" with `2` in the "Value" field. Repeat this by changing the in-game value to `1` and scanning for `1`. This process helps isolate the correct memory address.
  </Step>
  <Step num="5" title="Set New Player Limit">
    Once you have a few results, double-click them to move them to the bottom list. Change their values to `16`. This will set your lobby to a 16-player maximum. You can now close Cheat Engine.
  </Step>
</Steps>

<Callout type="tip">
After changing the values in Cheat Engine, you can close the program. The changes will persist for that game session. However, you'll likely need to repeat these steps every time you launch the game, as the memory addresses can change.
</Callout>

While playing with more friends might sound appealing, increasing the player count in Escape the Backrooms can significantly alter the game's intended experience and performance.

### Performance Considerations

The game's engine, Unreal Engine, has a technical limit for active players that, when pushed, can strain system resources. More players mean more data to process, which can lead to:

- **Lower Frame Rates**: Especially in visually complex areas or during intense entity encounters.
- **Increased Latency**: Server performance can degrade with a higher number of connected clients.
- **Game Instability**: Higher player counts might trigger bugs or crashes that are not present in standard 4-player lobbies.

| Player Count | Performance Impact | Server Stability | Horror Atmosphere | Team Coordination |
| :----------- | :----------------- | :--------------- | :---------------- | :---------------- |

### Altered Atmosphere

Escape the Backrooms thrives on its psychological horror elements, which are often amplified by the feeling of isolation and vulnerability. With more players:

- **Reduced Fear Factor**: The presence of many teammates can diminish the sense of dread and loneliness.
- **Increased Chaos**: Large groups might turn the horror experience into a more chaotic, less scary adventure.
- **Proximity Chat Challenges**: While proximity chat is a feature, managing communications with many players can become difficult, especially when trying to stay quiet.
