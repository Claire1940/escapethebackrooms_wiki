<Section icon={Sparkles} title="Essential Driver and Software Updates">
Outdated drivers, especially for your graphics card, are a leading cause of fatal errors in many games, including Escape the Backrooms. Keeping your system software current is crucial for game stability and performance.

<Steps>
  <Step num="1" title="Update Graphics Card Drivers">
    Open **Device Manager** by typing it into the Windows search bar. Expand "Display adapters," right-click your graphics card (e.g., NVIDIA, AMD, Intel), and select "Update driver." Choose "Search automatically for drivers." If this doesn't find the latest version, visit your GPU manufacturer's website (NVIDIA, AMD, Intel) to download and install the most recent drivers manually.
  </Step>
  <Step num="2" title="Update Windows OS">
    Navigate to **Settings > Update & Security > Windows Update**. Click "Check for updates" and install any pending updates. Restart your PC after the installation is complete. Keeping your Windows OS up-to-date ensures compatibility with the latest game technologies and bug fixes.
  </Step>
  <Step num="3" title="Install/Update Microsoft Visual C++ and DirectX">
    Many games rely on specific versions of Microsoft Visual C++ Redistributables and DirectX. If these are missing or corrupted, fatal errors can occur. Download the latest versions from Microsoft's official website. Install all three available Visual C++ files (x86 and x64 for different years) if prompted, and run the DirectX Web Installer to ensure all necessary components are present.
  </Step>
</Steps>

<Callout type="tip" title="Driver Update Checklist">
After updating drivers, always restart your computer. This ensures the new drivers are properly initialized and can prevent further conflicts.
</Callout>

<Section icon={Swords} title="System-Level Conflict Resolution">
Sometimes, other applications or system settings can interfere with Escape the Backrooms, leading to fatal errors. Identifying and mitigating these conflicts is crucial.

  <Card title="Manage Background Processes">
    - Open **Task Manager** (Ctrl+Shift+Esc).
    - Go to the "Processes" tab.
    - End any unnecessary background applications that consume significant CPU or RAM.
    - Specifically check for any "Escape the Backrooms" processes that might be stuck.
  </Card>
  <Card title="Antivirus and Firewall Exclusions">
    - Open **Windows Security** (or your third-party antivirus).
    - Go to "Virus & threat protection" > "Manage settings" > "Add or remove exclusions."
    - Add the entire game folder for Escape the Backrooms as an exclusion.
    - For Firewall, search "Allow an app through Windows Firewall," click "Change settings," and ensure Escape the Backrooms is allowed for both Private and Public networks.
  </Card>
</CardGrid>

| Check | Method | Purpose |
| :---- | :----- | :------ |
