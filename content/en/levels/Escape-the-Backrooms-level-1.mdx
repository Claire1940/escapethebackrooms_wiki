<Section icon={Target} title="Part 1 & 2: Elevators and the Key Hunt">

The first challenge in **Escape the Backrooms level 1** is a logic puzzle. You will find an elevator locked behind a color-coded panel. To find the combination, navigate to the "Car Room," which contains four vehicles of varying colors.

<CardGrid cols={2}>
  <Card title="The Car Puzzle">
    Read the car colors from **left to right**. Common combinations include Red, Red, Blue, Yellow, but these can vary. Input this sequence into the elevator to descend to Floor 2.
  </Card>
  <Card title="The Key Objective">
    On Floor 2, you must locate **four keys**. These are hidden randomly in cabinets, desk drawers, and lockers. The map loops, so keep a mental note of which rooms you have cleared.
  </Card>
</CardGrid>

  </Step>
  <Step num="3" title="Unlock Floor 2">

<Section icon={Shield} title="Part 3: The Three-Floor Maze">

Part 3 is one of the most frustrating sections for new players. It consists of three identical floors (-1, -2, and -3). Your goal is to find a specific door where an elevator has a random chance to spawn.

**Floor Breakdown:**
- **Floor -1**: Generally safe with no entities. Good for stocking up on supplies.
- **Floor -2**: One Skin-Stealer roams the halls. High elevator spawn probability.
- **Floor -3**: Extremely dangerous. Multiple entities may spawn. Only descend if the elevator isn't on higher floors.

<Callout type="info" title="Elevator RNG">
The elevator spawn is randomized per session. If you find a locked door that won't open, that was a potential spawn point that failed. Move to the next floor immediately.
</Callout>

<Section icon={Sparkles} title="Part 4: The Hub and MEG Base Override">

To progress through the final gate of **Escape the Backrooms level 1**, you must perform a security override. This requires entering **The Hub**, a central inter-dimensional zone.

**Terminal Puzzle Steps:**
1. Locate the security terminal outside the MEG Base.
2. Match the code on the right side of the screen with the corresponding column on the left.
3. Once inside, flick all four light switches on the right-hand wall.
4. Log into the computer using the password found on the nearby note (usually "**it heard you**").
5. Execute `gate.exe` to open the path back in the main level.

<Callout type="success" title="MEG Base Benefits">
The MEG Base is a safe zone. Use this time to replenish your sanity and stamina without the threat of entities.
</Callout>

| Terminal Column | Code Match Requirement | Action |
| :--- | :--- | :--- |
| **Column A** | LD (HL) | Select if right-side code matches |
| **Column B** | RX (02) | Select if right-side code matches |
| **Column C** | TZ (99) | Select if right-side code matches |

<Section icon={Swords} title="Part 5: Smilers and the Final Exit">

The final stretch of the level is a test of timing and nerves. You will enter a hallway where the lights flicker rhythmically. When the lights are on, you are safe. When they turn off, **Smilers** spawn instantly.

**Survival Rules:**
- **The Flicker**: When the lights flicker, you have exactly **3 seconds** to reach a safe zone.
- **Safe Zones**: These are small rooms with blue double doors.
- **The Exit**: Behind one of the blue doors in a safe zone is the transition to **Pipe Dreams**.

<CardGrid cols={3}>
  <Card title="The Smiler">
    Fast, aggressive, and lethal. They only appear in total darkness. Never get caught in the hallway when the lights cut.
  </Card>
  <Card title="Movement Strategy">
    Only move between one set of doors at a time. Do not try to sprint through the entire hallway in one light cycle.
  </Card>
  <Card title="Pipe Dreams Exit">
    The exit door looks identical to others but leads to a dark, pipe-filled corridor. This is your ticket out of Level 1.
  </Card>
</CardGrid>

<Callout type="warning" title="Sanity Check">
The darkness in Part 5 drains sanity at an accelerated rate. If your screen begins to distort, drink Almond Water immediately or you will succumb to the Smilers even in the "safe" zones.
</Callout>

</Section>
