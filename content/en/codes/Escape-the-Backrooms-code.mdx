| Object Type     | Associated Vending Machine | Counting Method                                  |

    Count the exact number of each object type present in the room.
  </Step>
  <Step num="3" title="Enter Numbers on Vending Machines">

### Level 5: Terror Hotel - Moth Jelly & Mailbox Puzzles
The elevator code for Level 5 is **314**. This level features two main puzzles.

#### Dumbwaiter & Moth Jelly
You need to collect Moth Jelly to obtain a key.

<Steps>
  <Step num="1" title="Find Note">
    Locate the Dumbwaiter in the left hallway of The Main Hall and read the note: "iF YOU FEED THEM THEY WiLL REWARD YOU".
  </Step>
  <Step num="2" title="Collect Moth Jelly">
    Using Bug Spray, search the entire floor for three clusters of Male Deathmoths. Spray them to collect Moth Jelly.
  </Step>
  <Step num="3" title="Use Dumbwaiter">
    Place one Moth Jelly into the Dumbwaiter and send it down. It will return with a key. Repeat this process three times to get three keys.
  </Step>
  <Step num="4" title="Open Glass Door">
    Use all three keys on the locks on the glass door to open a hallway leading to the Elevator.
  </Step>
</Steps>

<Callout type="warning" title="Limited Bug Spray">
Manage your Bug Spray carefully in Level 5. You'll need it for all three clusters of Male Deathmoths.
</Callout>

#### Mailbox & Room 235 Code
This puzzle involves collecting papers and deciphering a code.

| Puzzle Element | Description                                     |
| :------------- | :---------------------------------------------- |
| Mailbox        | Compartments for each main room, 7 open boxes randomized |
| Papers         | Found on drawers in rooms corresponding to open mailbox slots |
| Ceiling Code   | Appears after all papers are placed, spells "LOOK UP" |
| Room 235 Lock  | Number Lock with 3 possible codes               |

<Steps>
  <Step num="1" title="Collect Papers">
    Identify the 7 open compartments in the Mailbox. Travel to the corresponding rooms and collect the papers found on drawers.
  </Step>
  <Step num="2" title="Place Papers in Mailbox">
    Return to the Mailbox and place each collected paper into its correct compartment.
  </Step>
  <Step num="3" title="Reveal Ceiling Code">
    Once all papers are placed, a code will appear on the ceiling. The papers spell "LOOK UP" as a hint.
  </Step>
  <Step num="4" title="Enter Room 235 Code">
    Input the discovered code into the Number Lock on the Room 235 door. The possible codes are: **17564**, **05938**, or **89472**.
  </Step>
</Steps>
