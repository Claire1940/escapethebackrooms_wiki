<Section icon={Swords} title="コアゲームプレイメカニクスとサバイバル">

バックルームで生き残るには、ただ走るだけでなく、環境の手がかりに対する鋭い目と、エンティティの行動に対する深い理解が必要です。ゲームはプレイヤーをほとんどヒントなしで一連の相互接続されたレベルに投入し、出口を見つけ、鍵を集め、環境パズルを解くことを強います。コミュニケーションは不可欠であり、チームからあまりにも遠く離れると、孤立し、潜む敵意の手によって即座に死に至る可能性があります。

環境そのものがキャラクターです。例えば、レベル0では、黄色の壁紙とブーンという音を立てる光の繰り返しが、方向感覚を失わせるように設計されています。プレイヤーは、壁のしわやより深いセクションにつながる通気口など、「グリッチ」または異常を探す必要があります。ロープや鍵などのツールは散らばっていますが、その配置は初心者にはランダムに感じられるかもしれません。

| ツール/メカニクス | 目的 | 使用ヒント |
| :--- | :--- | :--- |
| **ロープ** | レベル移動 | 下のレベルに到達したり、経路をマークしたりするために使用します。 |
| **鍵** | ドアのロック解除 | 常に角や隠れた引き出しを確認してください。 |
| **VHSエフェクト** | 視覚的没入感 | 目の疲れや方向感覚の喪失を引き起こす場合はオフにしてください。 |
| **カスタマイズ** | キャラクターID | 暗闇の中でチームメイトを識別するために異なる色を使用してください。 |
| **近接チャット** | コミュニケーション | エンティティとの遭遇時に調整するために不可欠です。 |
