Escape the Backroomsの各レベルには、異なる課題とエンティティが存在します。構造化されたアプローチをとることで、脱出の可能性を大幅に高めることができます。

### レベル0：ロビー (The Lobby)
これは最初の試練であり、色あせた黄色い壁が続く終わりのない迷路です。目的は、壊れたはしごを見つけて修理し、鍵を入手して出口のロックを解除することです。数分後にはエンティティ「ハウラー」が現れ、追跡を開始します。

<Steps>
  <Step num="1" title="はしごのパーツを見つける">
    曲がりくねった廊下を探索し、散らばっているはしごのパーツを見つけます。それらを集めて修理しましょう。
  </Step>
  <Step num="2" title="鍵を入手する">
    修理したはしごを使って、鍵が置いてある高い場所の棚に到達します。
  </Step>
  <Step num="3" title="ロックされたドアを見つける">
    ロックされたドアを探します。このドアは通常、四角い落とし穴がある部屋へと続いています。
  </Step>
  <Step num="4" title="落とし穴を渡る">
    バランスを崩さないように注意しながら落とし穴を渡ります。落下すると、別の出口がある第2層に送られます。
  </Step>
</Steps>

<Callout type="warning" title="ハウラーに注意">
ハウラーは数分後にマップ上に出現し始めます。適切に対処しないと、壁越しにプレイヤーを見つけることもあります（マップ外からの観察による）。距離を保ち、隠れ場所を利用しましょう。
</Callout>

### レベル1：居住可能ゾーン (The Habitable Zone)
このレベルには広い駐車場があり、スキン・スティーラーが登場します。エレベーターのコードを見つけ、4つの鍵を集めて金庫のドアを開ける必要があります。

| フェーズ | 目的 | エンティティ | ヒント |
| :------------------- | :--------------------------------------------- | :------------------- | :------------------------------------------------------------------- |
| **駐車場** | エレベーターのコードを見つける（色付きの車から）。 | なし（初期状態） | 車の位置に注目してください。それがコードを示しています。 |
| **エレベーターの下** | 部屋に散らばっている**4つの鍵**を集める。 | **スキン・スティーラー** | スキン・スティーラーはドアを開けられません。回避のためにドアを活用しましょう。 |
| **暗いガレージ** | 次のエレベーターへ進む。 | **スキン・スティーラー** | 暗闇では発見が困難です。音に耳を澄ませてください。 |

### レベル進行の流れ
ゲームはセミリニアな進行をしており、しばしば「ザ・ハブ」に戻ったり、代替ルートが登場したりします。

| レベル名 | 主な目的 | 接続ポイント |
| :--------------------- | :-------------------------------------------------- | :------------------------------------------------------ |
| **レベル0（ロビー）** | 鍵を見つけ、出口を開ける。 | レベル1へ。 |
| **レベル1（居住可能ゾーン）** | エレベーターのコードを探し、鍵を集め、金庫を開ける。 | ザ・ハブへ。 |
| **ザ・ハブ** | M.E.G.拠点にアクセスし、ガレージのドアを開ける。 | パイプ・ドリーム1（出口A）またはレベル922（出口B）へ。 |
| **電気ステーション** | 9つのヒューズボックスを起動する。 | 放棄されたオフィスへ。 |
| **放棄されたオフィス** | 自販機パズルを解き、カメラを避ける。 | 恐怖のホテルへ。 |
| **恐怖のホテル** | モスゼリーを入手し、書類を集め、南京錠を解く。 | パイプ・ドリーム2へ。 |
| **パイプ・ドリーム2** | トンネルを進み、スマイラーの追跡から逃れる。 | レベル・ファンへ。 |
| **レベル・ファン** | パーティゴーヤーの目を盗んで進む。 | プールルームへ。 |
| **プールルーム** | 廊下を探索し、出口を見つける。 | 命がけの疾走へ。 |
| **命がけの疾走** | 病院の廊下を全力疾走し、障害物を避ける。 | レベル・ジ・エンドへ。 |
| **レベル・ジ・エンド** | VHSテープを集め、コンピュータを使用する。 | エッシャーのだまし絵の階段へ。 |
| **エッシャーの階段** | 94階まで登り、中央から飛び降りる。 | レベル94：モーションへ。 |
| **レベル94：モーション** | 浮遊する城に到達し、アニメーション・キングを倒す。 | 消灯へ。 |
| **消灯** | LIDARスキャナーで暗い迷路を進む。 | 海洋恐怖症（出口A）またはキティの家（出口B）へ。 |
| **海洋恐怖症** | 水没した村を通り、洞窟システムに潜る。 | レベル8へ。 |
| **レベル8** | 蛾の洞窟を進み、スキン・スティーラーを避ける。 | レベル0.11へ。 |
| **レベル0.11 & 188** | 光のパズルを解き、チェーンソーを使い、レバーを引く。 | レベル9へ。 |
| **レベル9** | セキュリティ端末を起動し、レッチを治療する。 | レベル10へ。 |
| **レベル10** | 麦畑を進み、タービンを起動する。 | レベル3999（真のエンディング）へ。 |

### エンディングを理解する

| エンディングの種類 | 達成方法 | 意義 |
| :----------------- | :----------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------- |
| **エンディングA** | 通常の進行後、レベル55.1（エレベーター）をクリアする。 | 「標準的」な脱出ルート。 |
| **エンディングB** | 拡張M.E.G.拠点から「Overgrowth」を完了した後、グラスルーム（The Grassrooms）を通り抜ける。 | 独自のパズルがある、より困難な脱出。 |
| **エンディングC** | ザ・ハブからスロープを通ってレベル922（同じ部屋が続く場所）へ行き、94階まで登って飛び降りる。 | エンティティのいない隠された脱出。 |
| **エンディングD** | （ソースに情報なし。通常は特定のアクションや他のエンディングによって解放される）。 | しばしばユニークなシークエンスや隠しエリアが関わる。 |
| **真のエンディング** | エンディングA、B、C、Dをすべて集め、レベル3999でアーケードマシンをプレイする。 | フロントルーム（現実世界）へ帰還する究極の脱出。 |

<Callout type="info" title="拡張M.E.G.拠点">
エンディングAを完了すると、拡張M.E.G.拠点にアクセスできるようになります。このエリアは新しいレベルや代替ルートに接続されており、さらなる挑戦と他のエンディングへの道を提供します。
