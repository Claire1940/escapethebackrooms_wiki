<Section icon={Map} title="Escape the Backroomsにおける敵の分類">

リミナルスペースで生き残るには、各レベルに潜む様々な**Escape the Backroomsの敵**を深く理解する必要があります。これらの実体は単なる障害物ではなく、特定の感覚トリガーを持つ知的な捕食者です。視覚で狩るもの、聴覚で狩るもの、あるいは生息するレベルの特定の仕掛けに結びついているものもいます。これらの脅威の階層を理解することが、出口に到達するための第一歩です。

2026年のメタにおいて、プレイヤーは実体をエンゲージメントスタイルに基づいて「ストーカー（追跡者）」、「チェイサー（猛追者）」、「環境ハザード」に分類しています。フェイスレス（Faceling）のようなストーカーは遠くからついてきますが、ハウラー（Howler）やクランプ（Clump）のようなチェイサーは、視認した瞬間に高速追跡を開始します。

| 実体名 | 主要レベル | 脅威レベル | 最善の防御策 |
| :--- | :--- | :--- | :--- |
| **スマイラー (Smiler)** | レベル 1, 2, ! | **極めて高い** | 明るい場所に留まる / 目を合わせない |
| **ハウンド (Hound)** | レベル 3, 5 | **高い** | 懐中電灯で怯ませる / 目を合わせる |
| **メス・デスモス (Female Deathmoth)** | レベル 5, 8 | **極めて高い** | しゃがみ歩き / ステルス |
| **ハウラー (Howler)** | レベル 0, 0.11 | **中程度** | 走る / 通気口に隠れる |
| **フェイスレス (Faceling)** | レベル 10 | **低〜中程度** | 頻繁な背後確認 / 走り抜ける |
| **パーティーポーパー (Partypooper)** | レベル 52 | **なし** | 案内に従ってついていく |

<Callout type="tip" title="プロのヒント">
常にスタミナバーを優先してください。ほとんどの実体はスプリントを効果的に管理すれば逃げ切ることができますが、スマイラーのいる廊下で息切れすることは即ゲームオーバーを意味します。
</Callout>

</Section>

<Section icon={Swords} title="高脅威の実体と回避戦術">

最も危険な**Escape the Backroomsの敵**に対処する場合、直接的なアプローチが成功することは稀です。実体固有の弱点に基づいて動きを適応させる必要があります。例えば、スマイラーは闇の達人ですが、物理的に影に縛られています。逆に、アニメイテッド・キングは、彼を寄せ付けないために懐中電灯を積極的に当てる必要があります。

<CardGrid cols={3}>
  <Card title="スマイラー (The Smiler)">
    - **行動**: 真っ暗なエリアに生息。
    - **弱点**: 明るく照らされたゾーンには入れない。
    - **戦略**: 近くにいるときは検知を避けるため懐中電灯を消し、追われた場合は固定された光源のある場所へ逃げ込む。
  </Card>
  <Card title="ハウンド (The Hound)">
    - **行動**: 四足歩行で追いかけてくる犬のような人型実体。
    - **弱点**: 直接的なアイコンタクトと強い光に怯む。
    - **戦略**: 懐中電灯を直接当てて一時的にスタンさせ、その隙に視線を切る。
  </Card>
  <Card title="メス・デスモス (Female Deathmoth)">
    - **行動**: 非常に攻撃的な大型の飛行昆虫。
    - **弱点**: 垂直方向の追跡能力が低い。
    - **戦略**: 物音を最小限にするためにしゃがみ歩きをする。レベル5では蒸気パイプを利用して、蛾との間に障壁を作る。
  </Card>
</CardGrid>

これらの実体との対話は、しばしば「だるまさんが転んだ」のようなゲームになります。レベル94の**アニメイテッド・キング**の場合、ルールは完全に変わります。暗い城内を進む間、彼の攻撃を阻止するために光を当て続けなければなりません。目を離すと彼は消え、別の角度から襲いかかる準備をします。

| アクション | 有効な相手 | 逆効果な相手 |
| :--- | :--- | :--- |
| **懐中電灯を点ける** | ハウンド、アニメイテッド・キング | スマイラー、デスモス |
| **しゃがみ歩き** | デスモス、レッチ | スマイラー、クランプ |
| **スプリント（全力疾走）** | ハウラー、フェイスレス | ハウンド（彼らの方が速い） |
| **隠れる（ベッドの下など）** | アニメーションズ、スクラッチャー | ハイドロリティス・プレイグ |

<Callout type="warning" title="危険地帯">
レベル ! （「命がけの疾走」）のシークエンスでは、ステルスは不可能です。完璧なスプリントを維持し、障害物を避けなければなりません。背後のクランプには速度制限がなく、一度でもつまずけば捕まってしまいます。
</Callout>

</Section>

<Section icon={Target} title="レベル 94 ステップバイステップ生存ガイド">

レベル 94は、アニメイテッド・キングとアニメーションズ（Animations）という2種類の異なる**Escape the Backroomsの敵**が登場するため、ユニークな挑戦となります。このレベルの昼夜サイクルが生存戦略を決定します。日中、家の中は安全ですが、夜が更けると人形たち（アニメーションズ）が狩りを始めます。

<Steps>
  <Step num="1" title="日中の準備">
    昼のサイクルの間に屋外エリアを偵察します。どの家にベッドや隠れられるクローゼットがあるかを確認しておきましょう。キッチンエリアで見つかるアーモンドウォーターや物資を回収します。
  </Step>
  <Step num="2" title="夜を生き延びる">
    日が沈んだら、すぐに家に入ってください。廊下に留まってはいけません。ベッドの下やクローゼットの中に隠れます。アニメーションズは姿を見られると家の中に侵入してくるため、音楽が変わるか日が昇るまで隠れ続けてください。
  </Step>
  <Step num="3" title="城への潜入">
    城に到達したら、アニメイテッド・キングに備えてください。彼はピエロの姿で現れ、暗闇の中であなたをストーキングします。懐中電灯のバッテリーを充電しておきましょう。
  </Step>
  <Step num="4" title="キングを阻止する">
    ジェットコースターの出口に向かって城内を進む際、頻繁に後ろを振り返ってください。キングが見えたらライトを当てます。彼は凍り付くか消滅し、数秒間の前進する猶予を与えてくれます。
  </Step>
  <Step num="5" title="最終脱出">
    部屋の端にあるジェットコースターに到達したらすぐに乗り込みます。キングは乗り物まで追ってくることはできません。これがレベル 94の出口となります。
  </Step>
</Steps>

<Callout type="success" title="安息の地">
レベル 52のパーティーポーパーは、数少ない敵対的でない実体の一つです。逃げる必要はありません。彼についていけば、危険な通路を回避してレベルの出口まで直接案内してくれます。
</Callout>

</Section>

<Section icon={Shield} title="環境実体とステルスメカニクス">

バックルームの脅威のすべてが人型であるとは限りません。プールルーム（レベル 37）で見られる**ハイドロリティス・プレイグ（Hydrolitis Plague）**のような液体ベースのハザードもあれば、特定の廊下を支配する巨大な蜘蛛のような実体、**アラネア・メンブリ（Aranea Membri）**もいます。これらを生き抜くには、戦闘や逃走だけでなく環境への意識が必要です。

「The End」レベルの**スクラッチャー（Scratcher）**は、音に敏感な実体の典型例です。細長い手足で図書室を巡回しています。音を立てすぎたり、長時間視界に入っていたりすると襲いかかってきます。しかし、図書室には火災報知器が設置されています。これをおとりとして使い、スクラッチャーを部屋の反対側に誘い出すことで、安全に必要なテープを探すことができます。

| 実体 | 固有のメカニクス | 環境による対抗策 |
| :--- | :--- | :--- |
| **ボーン・シーフ** | 暗い穴に隠れる | ゲートに電力を供給して進路を塞ぐ |
| **ザ・シング** | 水中の捕食者 (L7) | 常にボートの中に留まる |
| **アラネア・メンブリ** | 廊下の巡回者 | 脇の部屋に逃げ込み、ドアを閉める |
| **ハイドロリティス・プレイグ** | 意志を持つ液体 | 接触を避ける。テーブルの下に隠れる |

<Callout type="info" title="メカニクスのアップデート">
2026年のアップデートにより、実体のパスファインディング（経路探索）が向上しました。部屋に入るのを見られた場合、以前よりも頻繁にベッドの下を確認するようになっています。実体がドアを入る前に、完全に隠れていることを確認してください。
</Callout>

</Section>
