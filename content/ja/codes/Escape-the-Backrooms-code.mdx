### レベル1：居住可能ゾーン - 鍵と金庫

レベル1では、より複雑なメカニズムが導入され、複数の鍵を見つけ、時間制限のある金庫パズルを解く必要があります。このレベルはいくつかのフロアに分かれており、それぞれに独自の課題があります。

#### フロア2：4つの鍵
このフロアはフロア1よりもかなり広いです。あなたの目的は、さまざまな部屋に散らばっている**4つの鍵**を見つけることです。

#### フロア2：金庫のドアパズル
4つの鍵をすべて集めたら、**金庫のドア**がある部屋を見つけます。このパズルは素早い行動を必要とします。

<Steps>
  <Step num="1" title="すべての鍵を挿入する">
    金庫のドアの周りの対応する鍵穴に、4つの鍵をそれぞれ挿入します。
  </Step>
  <Step num="2" title="鍵を同時に回す">
    すべての鍵を同時に回します。短時間で自動的に鍵が戻ってしまうので、素早く行います。
  </Step>
  <Step num="3" title="フロア3にアクセスする">
    すべての鍵が同時に回されれば、金庫のドアが開き、フロア3へのアクセスが許可されます。
  </Step>
</Steps>

<Callout type="warning" title="タイミングが重要">
レベル1のフロア2にある金庫のドアパズルは、正確なタイミングが必要です。繰り返し試行を避けるために、素早く鍵を回す練習をしてください。
</Callout>

#### フロア3：とらえどころのないエレベーター
フロア3は薄暗く、3つの個別のサブフロアで構成されています。あなたの目標は、フロア4へのエレベーターを見つけることです。

| 特徴          | 説明                                               |
| :-------------- | :------------------------------------------------- |
| 構造          | 3つの個別のフロア、同様の基本的なレイアウト      |
| エレベーターの出現 | サブフロアごとに1つの可能性のある場所、ランダム化 |
| ロックされたドア | 現在のサブフロアでアクティブ化しなかった潜在的なエレベーターの出現を示す |

</Section>

### レベル4：廃オフィス - 自動販売機パズル
レベル4のエレベーターコードは**729**です。主なパズルは自動販売機とオブジェクトのカウントに関するものです。

<Callout type="success" title="パズル解決済み">
自動販売機パズルは、部屋の中の特定のオブジェクトを正確に数えることに依存します。数字を入力する前に、数え間違いがないか再確認してください！
</Callout>

| オブジェクトの種類 | 関連する自動販売機 | カウント方法                                     |
| :-------------- | :----------------- | :----------------------------------------------- |
| 椅子            | 自動販売機1        | 部屋のすべての椅子を数える                       |
| テーブル        | 自動販売機2        | 部屋のすべてのテーブルを数える                   |
| ウォータークーラー | 自動販売機3        | 部屋のすべてのウォータークーラーを数える         |
| 本の山          | 自動販売機4        | 部屋のすべての目に見える本の山を数える           |

<Steps>
  <Step num="1" title="オブジェクトの種類を識別する">
    4つの自動販売機がある部屋を探します。各自動販売機は、椅子、テーブル、ウォータークーラー、本の山という特定のオブジェクトの種類に対応しています。
  </Step>
  <Step num="2" title="オブジェクトを数える">
    部屋にある各オブジェクトの正確な数を数えます。
  </Step>
  <Step num="3" title="自動販売機に数字を入力する">
    各自動販売機で、関連するオブジェクトの数に対応する数字ボタンを選択します（オブジェクトボタン自体ではありません）。
  </Step>
  <Step num="4" title="ドアのロックを解除する">
    すべての値が正しければ、次のエリアに通じるドアのロックが解除されます。
  </Step>
</Steps>

### レベル5：テラーホテル - 蛾のゼリーと郵便受けのパズル
レベル5のエレベーターコードは**314**です。このレベルには2つの主要なパズルがあります。

#### ダムウェイターと蛾のゼリー
鍵を手に入れるには、蛾のゼリーを集める必要があります。

<Steps>
  <Step num="1" title="メモを見つける">
    メインホールの左廊下にあるダムウェイターを見つけ、「iF YOU FEED THEM THEY WiLL REWARD YOU」（もしあなたが彼らに餌を与えれば、彼らはあなたに報いるだろう）というメモを読みます。
  </Step>
  <Step num="2" title="蛾のゼリーを集める">
    殺虫スプレーを使って、フロア全体で3つの雄のデスモス（Male Deathmoths）の群れを探します。それらをスプレーして蛾のゼリーを集めます。
  </Step>
  <Step num="3" title="ダムウェイターを使う">
    蛾のゼリーを1つダムウェイターに入れて下に送ります。すると、鍵が戻ってきます。このプロセスを3回繰り返して、3つの鍵を手に入れます。
  </Step>
  <Step num="4" title="ガラスのドアを開ける">
    ガラスのドアの鍵穴に3つの鍵すべてを使って、エレベーターに通じる廊下を開きます。
  </Step>
</Steps>

<Callout type="warning" title="殺虫スプレーの制限">
レベル5では、殺虫スプレーを慎重に管理してください。3つの雄のデスモスの群れすべてに必要になります。
</Callout>

#### 郵便受けと部屋235のコード
このパズルは、書類を集めてコードを解読するものです。

| パズル要素     | 説明                                           |
| :------------- | :--------------------------------------------- |
| 郵便受け       | 各メインルーム用の区画、7つの開いた箱はランダム化 |
| 書類           | 開いた郵便受けのスロットに対応する部屋の引き出しで見つかる |
| 天井のコード   | すべての書類が配置されると現れ、「LOOK UP」と表示される |
| 部屋235の鍵    | 3つの可能なコードを持つ数字ロック             |

<Steps>
  <Step num="1" title="書類を集める">
    郵便受けの中にある7つの開いた区画を特定します。対応する部屋に行き、引き出しにある書類を集めます。
  </Step>
  <Step num="2" title="書類を郵便受けに入れる">
    郵便受けに戻り、集めた書類をそれぞれの正しい区画に入れます。
  </Step>
  <Step num="3" title="天井のコードを明らかにする">
    すべての書類が配置されると、天井にコードが現れます。書類はヒントとして「LOOK UP」と表示されます。
  </Step>
  <Step num="4" title="部屋235のコードを入力する">
    発見したコードを部屋235のドアにある数字ロックに入力します。可能なコードは、**17564**、**05938**、または**89472**です。
  </Step>
</Steps>

レベル8のエレベーターコードは**826**です。このコードは、複雑でしばしば方向感覚を失わせる洞窟ネットワークをナビゲートするために非常に重要です。探索中は、狭い空間や行き止まりに備えてください。
