<Section icon={Sparkles} title="必須ドライバーとソフトウェアの更新">
古いドライバー、特にグラフィックカードのドライバーは、Escape the Backrooms を含む多くのゲームで致命的なエラーの主な原因となります。システムソフトウェアを最新の状態に保つことは、ゲームの安定性とパフォーマンスにとって非常に重要です。

<Steps>
  <Step num="1" title="グラフィックカードドライバーの更新">
    Windows の検索バーに「**デバイスマネージャー**」と入力して開きます。「ディスプレイアダプター」を展開し、グラフィックカード（例：NVIDIA、AMD、Intel）を右クリックして「ドライバーの更新」を選択します。「ドライバーを自動的に検索」を選択します。これで最新バージョンが見つからない場合は、GPU メーカーのウェブサイト（NVIDIA、AMD、Intel）にアクセスして、最新のドライバーを手動でダウンロードしてインストールしてください。
  </Step>
  <Step num="2" title="Windows OS の更新">
    **設定 > 更新とセキュリティ > Windows Update** に移動します。「更新プログラムのチェック」をクリックし、保留中の更新プログラムをすべてインストールします。インストールが完了したら PC を再起動します。Windows OS を最新の状態に保つことで、最新のゲームテクノロジーとの互換性とバグ修正が保証されます。
  </Step>
  <Step num="3" title="Microsoft Visual C++ と DirectX のインストール/更新">
    多くのゲームは、特定のバージョンの Microsoft Visual C++ 再頒布可能パッケージと DirectX に依存しています。これらが不足しているか破損している場合、致命的なエラーが発生する可能性があります。Microsoft の公式ウェブサイトから最新バージョンをダウンロードしてください。プロンプトが表示された場合は、利用可能なすべての Visual C++ ファイル（x86 および x64、異なる年式）をインストールし、DirectX Web インストーラーを実行して必要なすべてのコンポーネントが存在することを確認してください。
  </Step>
</Steps>

<Callout type="tip" title="ドライバー更新チェックリスト">
ドライバーを更新した後、必ずコンピューターを再起動してください。これにより、新しいドライバーが適切に初期化され、さらなる競合を防ぐことができます。
</Callout>

<Section icon={Target} title="安定性のためのゲーム固有のトラブルシューティング">
一般的なシステムメンテナンスに加えて、ゲームの設定またはファイル内の特定のアクションにより、致命的なエラーが解決されることがあります。これらの手順は、ゲーム自体が正しく設定され、ファイルが破損していないことを確認することに焦点を当てています。

| トラブルシューティングの手順 | 説明 | 期待される結果 |
| :------------------- | :------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------- |
| **ゲームファイルの確認** | Steam で Escape the Backrooms を右クリックし、**プロパティ > インストール済みファイル** に移動し、「ゲームファイルの整合性を確認」をクリックします。 | Steam は、破損または不足しているゲームファイルを確認し、必要なコンポーネントを再ダウンロードして、一般的なインストール関連のエラーを修正します。 |
| **管理者として実行** | ゲームの実行ファイルまたはショートカットを右クリックし、**プロパティ > 互換性** に移動し、「管理者としてこのプログラムを実行する」にチェックを入れます。 | ゲームの権限を昇格させ、特にファイルアクセスやシステムリソースに関する不十分な権限によって引き起こされる問題を防止します。     |
| **互換モード** | 同じ互換性タブで、ゲームを **Windows 7** または **Windows 8** の互換モードで実行してみてください。             | ゲームが最新の OS 機能と互換性の問題がある場合、新しい Windows バージョンでの問題を解決できます。                          |
| **フルスクリーン最適化の無効化** | 同じ互換性タブで、「フルスクリーン最適化を無効にする」にチェックを入れます。                                                                | ゲームがフルスクリーンモードで実行されるときに、ディスプレイ関連のクラッシュやパフォーマンスの問題を防ぐことができます。                                          |
| **グラフィック設定の調整** | ゲーム内またはグラフィックカードのコントロールパネルから、ゲームを**ハイパフォーマンス**モードに設定します。                                    | 他のアプリケーションよりもゲームのパフォーマンスを優先し、リソース関連の致命的なエラーの可能性を低減します。                        |

</Section>

<Section icon={Swords} title="システムレベルの競合解決">
場合によっては、他のアプリケーションやシステム設定が Escape the Backrooms と干渉し、致命的なエラーを引き起こすことがあります。これらの競合を特定し、軽減することが重要です。

<CardGrid cols={2}>
  <Card title="バックグラウンドプロセスの管理">
    - **タスクマネージャー**（Ctrl+Shift+Esc）を開きます。
    - 「プロセス」タブに移動します。
    - 大量の CPU または RAM を消費している不要なバックグラウンドアプリケーションを終了します。
    - 特に、スタックしている可能性のある「Escape the Backrooms」プロセスを確認します。
  </Card>
  <Card title="ウイルス対策とファイアウォールの除外">
    - **Windows セキュリティ**（またはサードパーティのウイルス対策ソフト）を開きます。
    - 「ウイルスと脅威の防止」>「設定の管理」>「除外の追加または削除」に移動します。
    - Escape the Backrooms のゲームフォルダ全体を除外に追加します。
    - ファイアウォールの場合、「Windows ファイアウォールを介したアプリの許可」を検索し、「設定の変更」をクリックして、Escape the Backrooms がプライベートネットワークとパブリックネットワークの両方で許可されていることを確認します。
  </Card>
</CardGrid>

<Callout type="info">
**ヒント:** ゲームを起動する*前に*ウイルス対策ソフトまたはファイアウォールを一時的に無効にすると、それらが致命的なエラーの原因であるかどうかを特定するのに役立ちます。セキュリティのため、その後で再度有効にすることを忘れないでください。
</Callout>

### 高度なシステムチェック

| チェック | 方法 | 目的 |
| :---- | :----- | :------ |
| **SFC スキャン** | コマンドプロンプトを管理者として開き、「`sfc /scannow`」と入力して Enter キーを押します。 | 不安定性の原因となる可能性のある破損した Windows システムファイルをスキャンして復元します。 |
| **クリーンブート** | MSConfig を使用してクリーンブートを実行し、すべての非 Microsoft サービスとスタートアップ項目を無効にします。 | 最小限のドライバーとプログラムで Windows を起動することで、ソフトウェアの競合を特定するのに役立ちます。 |
| **ディスクチェック** | ゲームドライブを右クリックし、**プロパティ > ツール > チェック** に移動します。 | ゲームデータを破損させる可能性のある不良セクターやファイルシステムエラーを特定して修正します。 |
