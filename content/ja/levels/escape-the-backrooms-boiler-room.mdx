<Section icon={Shield} title="不可欠なサバイバルツールとメカニクス">

**バックルームからの脱出ボイラールーム**を生き残るには、セクション開始時に提供されるツールを習得する必要があります。ルーム235に入り、階段を上ると、温度計が置かれたテーブルがあります。これらは任意ではなく、暗く蒸気に満ちた廊下で脅威が見えるようになる前に脅威を検出する主要な手段です。

<CardGrid cols={3}>
  <Card title="温度計">
    - **安全な温度**: 72°Fは近くに差し迫った脅威がないことを示します。
    - **危険な温度**: 120°F以上に急上昇すると、部屋にメスのデスモスがいることを意味します。
    - **精度**: 曲がり角を曲がる前に角を確認するために使用します。
  </Card>
  <Card title="赤いバルブ">
    - **防御**: これらを回すと、高圧蒸気が噴出します。
    - **忌避剤**: 蒸気は蛾が越えられない物理的な障壁として機能します。
    - **タイミング**: 攻撃的な蛾に積極的に追いかけられているときに使用するのが最適です。
  </Card>
  <Card title="視覚的な手がかり">
    - **熱波**: 画面上の空気の歪みは極度の熱を示します。
    - **蛾の溜まり**: 床の蛾ゼリーは巣が近いことを示唆します。
    - **照明**: 巨大な蛾の近くで懐中電灯を使用しないでください。
  </Card>
</CardGrid>

温度のメカニズムは一貫しています。温度計が正確に72度を示している場合、あなたは「中立」ゾーンにいます。しかし、パイプの迷宮のような性質上、数秒で120度のゾーンに足を踏み入れる可能性があります。観察力のあるプレイヤーは、「きらめく」空気を探すこともできます。これは、温度計が隠れていても、近くに蛾がいることを視覚的に警告するものです。

| ツール | 主な用途 | 有効性 |
| :--- | :--- | :--- |
| **温度計** | 検出 | ★★★★★ |
| **懐中電灯** | ナビゲーション | ★★☆☆☆ (危険) |
| **蒸気バルブ** | 群衆制御 | ★★★★☆ |
| **しゃがむ** | ステルス | ★★★★★ |

<Callout type="tip" title="明るさ調整">
熱波が見にくい場合は、ゲーム内のガンマ設定を上げてみてください。これにより、ボイラールームの暗い石壁に対して空気の歪みがより目立つようになります。
</Callout>

<Section icon={Target} title="デスモスの迷宮をナビゲートする">

**バックルームからの脱出ボイラールーム**をナビゲートすることは、忍耐力が試されます。メスのデスモスは巨大で、しばしば天井全体を占めたり、狭い廊下を塞いだりします。ホテルの小さなオスの蛾とは異なり、これらのエンティティは、立っているときや走っているときに検出されると、すぐに魂を食い尽くします。

<Steps>
  <Step num="1" title="最初の入場">
    テーブルから温度計を拾います。最初の廊下を確認し、72度であれば、最初の分岐点まで進みます。
  </Step>
  <Step num="2" title="最初の遭遇">
    温度計が120°Fに跳ね上がったら、すぐに**コントロール**（しゃがみ）キーを押し続けます。しゃがんでいる限り、巨大な蛾は、たとえ真下を歩いても、通常はあなたを傷つけることはありません。
  </Step>
  <Step num="3" title="パイプに沿って進む">
    天井と壁に沿って大きな工業用パイプをたどります。これらはしばしばレベルの中心と最終的な出口へと導きます。
  </Step>
  <Step num="4" title="バルブ防御">
    誤って立ち上がって蛾の攻撃を誘発してしまった場合は、最も近い赤いバルブに向かって走ります。それを操作して蒸気を放出し、蛾を後退させ、その巡回をリセットさせます。
  </Step>
</Steps>

迷路には3つの異なるチェックポイントがあります。死亡した場合、最後に通過したチェックポイントでリスポーンします。これらは通常、小さなテーブルやランタンでマークされています。これらをランドマークとして使用し、迷子になるのではなく、正しい方向に進んでいることを確認してください。

| エンティティタイプ | 行動 | 対策 |
| :--- | :--- | :--- |
| **オスのデスモス** | 防御的/群がる | 虫除けスプレー/蛾ゼリー |
| **メスのデスモス** | 非常に攻撃的 | しゃがむ/蒸気バルブ |
| **ハウンド/スキン・スティーラー** | 徘徊（ホテルのみ） | クローゼット/ベッドに隠れる |

<Callout type="success" title="しゃがみメカニクス">
身長は「しゃがんだ」状態ほど重要ではないという既知の戦略（おそらくグリッチ）があります。パイプの上でわずかに高くても、しゃがみボタンを押し続けている限り、蛾はあなたを無視することがよくあります。
</Callout>

</Section>

**バックルームからの脱出ボイラールーム**は、「偽のドア」で有名です。このレベルには、出口と全く同じに見えるドアが散りばめられていますが、間違ったドアを開けると即死につながる可能性があります。これらのドアのほとんどは、デスモスの巣がぎっしり詰まった小さな部屋に通じています。一度開くと、中の蛾がプレイヤーに群がり、逃げる余地はほとんど残りません。

巨大なメスのデスモスがここに主な住民です。ボイラールーム全体に約9匹が散らばっています。一部は静止しており、特定の廊下を守っていますが、他のものは限られた巡回経路を持っています。彼らの配置を理解することが、スピードランを成功させるか、安全に初めてクリアするための鍵となります。

| ドアの色/タイプ | マーカー | 結果 |
| :--- | :--- | :--- |
| **木製ドア** | なし | 行き止まり/トラップ |

| **出口ドア** | 緑色のサイン | レベルクリア |

<Callout type="warning" title="懐中電灯の敵対">
巨大な蛾は光に非常に敏感です。懐中電灯を直接頭に当てると、しゃがんでいるかどうかに関係なく、敵対的になります。代わりに周辺視野または温度計を使用してナビゲートしてください。

<Section icon={Trophy} title="レベル6への最終経路">

**バックルームからの脱出ボイラールーム**の最後の区間は、正気度が低い場合は「逃げ出す」精神で、時間がある場合はゆっくりと methodical な這い進みが必要です。出口は、最後の2回の蛾との遭遇を通過した後に現れる、長い一直線の廊下の終点にあります。

正しい道を見つけるには：
1. レベルの後半で床に沿って走るオレンジ色のパイプをたどります。
2. ランドマーク、特に大きな円筒形の容器を探します。
3. 緑色の光る**EXIT**サインのあるドアが見えたら、終点に到達しています。

このドアを開けると、すぐに**レベル6：消灯**に移動します。これは完全な暗闇と音によるナビゲーションに焦点を当てたレベルです。レベル6は精神状態にさらに負担をかけるため、通過する前にできるだけ正気度を回復していることを確認してください。

<Callout type="info" title="チェックポイント回復">
死亡してチェックポイントに戻っても、慌てないでください。ルートは見た目よりも短いです。キャニスターと赤いバルブをたどれば、60秒以内に元の軌道に戻れます。
