<Section icon={Map} title="ザ・ハブの理解：バックルームからの脱出 ロードレベル">
『Escape the Backrooms』のザ・ハブは、トンネルのような構造と相互接続性から「ロードレベル」と呼ばれることが多い、重要な中心エリアです。この大きな地下トンネルは、ゲーム内の他の多くのレベルへのアクセスを提供する拠点として機能します。ザ・ハブをうまくナビゲートすることは、それぞれのドアがロック解除された後に、プレイヤーが異なるゾーン間を移動できるようにするために不可欠です。

ザ・ハブに最初に入ったときの主な目的は、メインゲートを起動し、それによって他のエリアへのアクセスを許可することです。これには、トンネル内にある放棄されたM.E.G.基地と対話することが含まれます。

<Callout type="tip" title="ナビゲーションのヒント">
ザ・ハブのドアには常に注意を払ってください。他のレベルへのアクセスを提供しますが、ザ・ハブから使用する前に、それらの特定のレベル内からロックを解除する必要があります。それに応じて探索計画を立ててください。
</Callout>

<Section icon={Target} title="目的とゲートの起動">
ザ・ハブへの最初の進入では、重要なゲートを中心とした2つの目的があります。このゲートの起動は、バックルーム内でのさらなる探索機会を開くために不可欠です。

<Steps>
  <Step num="1" title="M.E.G.基地を見つける">
    レベル1から到着したら、最初のタスクはザ・ハブ内にある**M.E.G.基地**を見つけて入ることです。通常、メインのトンネルの左側にあります。
  </Step>
  <Step num="2" title="コンピューターのロックを解除する">
    M.E.G.基地内にはコンピューターがあります。ロックを解除するには、パスワード「**itheardyou**」を入力します。これにより、ファイルへのアクセスが許可されます。
  </Step>
  <Step num="3" title="ゲートを起動する">
    コンピューターのロックが解除されたら、「**Gate.exe**」という名前のファイルを開きます。このファイルを実行すると、ザ・ハブのメインゲートが正常にロック解除されます。
  </Step>
  <Step num="4" title="レベル1に戻る（オプション）">
    ゲートがロック解除されたら、レベル1に戻ることができます。以前アクセスできなかったゲートが開き、進行を続けることができます。
  </Step>
</Steps>

<Callout type="warning" title="パスワードのセキュリティ">
パスワード「itheardyou」は大文字と小文字を区別します。M.E.G.基地のコンピューターでアクセス問題が発生しないように、表示されているとおりに正確に入力してください。
</Callout>

</Section>

<Section icon={Swords} title="ザ・ハブからレベルのドアを解除する">
ザ・ハブには10の異なるドアがあり、それぞれが異なるレベルに通じています。ただし、これらのドアはすぐにはアクセスできません。まず、それぞれのレベル内からロックを解除する必要があります。このシステムは、プレイヤーがザ・ハブに簡単にアクセスできるようになる前に、各レベルを完全に探索することを促します。

いくつかの主要なレベルのドアを見つけてロックを解除するためのガイドを以下に示します。

| レベル番号 | ドアを解除する場所 | 主な特徴/ヒント |
| :----------- | :---------------------- | :----------------- |
| **レベル 0**  | レベル8から、マニラ室の近くの小さなエリア。 | マニラ室から前方に進み、少し右へ。 |
| **レベル 1**  | レベル1の4階。 | はっきりと目立ち、開かれた場所にあり、最初の進行に不可欠です。 |
| **レベル 4**  | 大きな柱の部屋の左側。 | 階段を上がった後、空の部屋を通り抜けて柱の部屋へ。 |
| **レベル 7**  | 2階のエリア、特定のドアを通って。 | 最初の部屋の上のドアを通り、さらに階段を上がります。 |
| **レベル 9**  | 特定の家、地下の石階段。 | 2階にレッチが守っている家を探します。 |
| **レベル 37** | 広い部屋の階段の頂上。 | 四角い入り口に入って、階段のある広い部屋を見つけます。 |
| **レベル 52** | 最初の部屋の右側。 | ドアは右側に目立つように配置されています。 |
| **レベル 94** | 最初の家の2階。 | 2階の部屋のドアを開けると、ザ・ハブのドアがあります。 |
| **レベル 188**| 多くのドアがラベル付けされた部屋で、「H」とラベル付けされたドアを通って。 | 「H」ドアに入った後、廊下を進みます。 |
| **レベル 3999**| 最初の左側にあるスタッフ専用ドアの後ろ。 | スライドドアを通り抜けてから、左側でスタッフアクセスを探します。 |

<Callout type="info" title="ドアのロック解除メカニズム">
ザ・ハブの各ドアは、目的地のレベルから起動されるまで一方通行のポータルとして機能します。これは、即座の入場ではなく、高速移動ポイントを確立するものと考えてください。
</Callout>
