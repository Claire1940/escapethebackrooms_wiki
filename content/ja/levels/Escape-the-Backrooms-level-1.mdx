<Section icon={Shield} title="パート3：3階建ての迷路">

パート3は、新規プレイヤーにとって最も苛立たしいセクションの一つです。これは3つの同一のフロア（-1、-2、-3）で構成されています。あなたの目標は、エレベーターがランダムに出現する特定のドアを見つけることです。

**フロアの内訳：**
- **フロア -1**：エンティティがおらず、一般的に安全です。物資を補充するのに適しています。
- **フロア -2**：スキン・スティーラーが1体、廊下を徘徊しています。エレベーターの出現確率が高いです。
- **フロア -3**：非常に危険です。複数のエンティティが出現する可能性があります。エレベーターが上の階にない場合にのみ降りてください。

<Callout type="info" title="エレベーターのRNG">
エレベーターの出現はセッションごとにランダムです。開かないロックされたドアを見つけた場合、それは失敗した潜在的な出現ポイントでした。すぐに次のフロアへ移動してください。
</Callout>

| フロアレベル | 危険度評価 | エンティティの存在 | 戦利品の密度 |
| :--- | :--- | :--- | :--- |
| **フロア -1** | ★☆☆☆☆ | なし | 高 |
| **フロア -2** | ★★★☆☆ | スキン・スティーラー | 中 |
| **フロア -3** | ★★★★★ | 複数のスキン・スティーラー | 低 |

</Section>

<Section icon={Sparkles} title="パート4：ハブとMEG基地のオーバーライド">

**Escape the Backrooms レベル1**の最後のゲートを進むには、セキュリティオーバーライドを実行する必要があります。これには、中心となる異次元ゾーンである**ハブ**に入る必要があります。

**ターミナルパズルの手順：**
1. MEG基地の外にあるセキュリティターミナルを見つけます。
2. 画面右側のコードを左側の対応する列と一致させます。
3. 中に入ったら、右側の壁にある4つの電気スイッチをすべて入れます。
4. 近くのメモ（通常「**it heard you**」）に記載されているパスワードを使用してコンピュータにログインします。
5. `gate.exe`を実行して、メインレベルに戻る経路を開きます。

<Callout type="success" title="MEG基地の利点">
MEG基地は安全地帯です。この時間を利用して、エンティティの脅威なしに正気度とスタミナを回復してください。
</Callout>

| ターミナル列 | コード一致要件 | アクション |
| :--- | :--- | :--- |
| **列A** | LD (HL) | 右側のコードが一致する場合に選択 |
| **列B** | RX (02) | 右側のコードが一致する場合に選択 |
| **列C** | TZ (99) | 右側のコードが一致する場合に選択 |

</Section>

<Section icon={Swords} title="パート5：スマイラーと最後の出口">

レベルの最終段階は、タイミングと神経の試練です。光がリズミカルに点滅する廊下に入ります。光が点灯しているときは安全です。光が消えると、**スマイラー**が瞬時に出現します。

**サバイバルルール：**
- **点滅**：光が点滅したら、安全地帯に到達するまで正確に**3秒**しかありません。
- **安全地帯**：青い両開きドアのある小さな部屋です。
- **出口**：安全地帯の青いドアの1つの向こうには、**パイプ・ドリーム**への移行があります。

<CardGrid cols={3}>
  <Card title="スマイラー">
    素早く、攻撃的で、致命的です。彼らは完全な暗闇の中でしか現れません。光が消えたときに廊下で捕まらないようにしてください。
  </Card>
  <Card title="移動戦略">
    一度に1組のドアの間だけを移動してください。1回の光のサイクルで廊下全体を走り抜けようとしないでください。
  </Card>
  <Card title="パイプ・ドリーム出口">
    出口のドアは他のドアと見た目は同じですが、暗くてパイプだらけの通路に通じています。これがレベル1から脱出するための切符です。
  </Card>
</CardGrid>

<Callout type="warning" title="正気度チェック">
パート5の暗闇は正気度を加速的に消耗させます。画面が歪み始めたら、すぐにアーモンドウォーターを飲んでください。そうしないと、「安全」なゾーンにいてもスマイラーにやられてしまいます。
</Callout>

</Section>
