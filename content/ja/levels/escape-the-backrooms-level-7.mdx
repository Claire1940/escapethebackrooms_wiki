<Section icon={Map} title="環境概要：深海恐怖症の海">

**Escape the Backrooms レベル7**、通称「深海恐怖症（Thalassophobia）」は、プレイヤーに圧倒的な、無限に広がる海を提示します。これまでのレベルの狭い廊下とは異なり、この環境はその広大さと深海の心理的な恐怖によって特徴づけられます。このレベルには、暗く濁った海水に浮かぶ、または部分的に水没した多くの郊外の家屋があります。海底は砂とギザギザの岩石で構成されており、水中セグメントで重要な目印となります。

軋む木材の音と遠くの水が動く音が重くのしかかる雰囲気です。レベル6から入ると、プレイヤーは初期の準備エリアとなる「スターティングハウス」にたどり着きます。このレベルは、ゲームの進行における主要な転換点であり、工業的でオフィスのような設定から、より超現実的で自然主義的な恐怖へと移行します。

| 特徴 | 説明 |
| :--- | :--- |
| **雰囲気** | 緊張感の高い、暗い海、濃い霧 |
| **難易度** | 中程度（ナビゲーション重視） |
| **主な危険** | 深海、The Thing、視界不良 |
| **主要なランドマーク** | スターティングハウス、灯台、シャークタンク |

<Callout type="tip" title="ナビゲーションのヒント">
海は広大で迷子になりやすいです。灯台が点灯するまでは、常にスターティングハウスまたは最も近い見える建造物を視界に入れておきましょう。
</Callout>

<Section icon={Target} title="水面ナビゲーション：光を追って">

**Escape the Backrooms レベル7**の前半は、水面で行われます。スターティングハウスを出た後、利用可能なボートを使って周囲の建造物を探索する必要があります。ここでのサバイバルは、資源管理と環境の変化を認識することにかかっています。

海を進むにつれて、やがて濃い霧が発生し、視界が遮られ、通常のナビゲーションが不可能になります。これは、あなたの忍耐力と観察力を試すために設計されたスクリプトイベントです。霧の中で数分待つと、遠くの家屋にある灯台が点灯します。この光が、最終的な家屋を見つける唯一の信頼できる方法です。

| アイテム | 場所 | 用途 |
| :--- | :--- | :--- |
| **アーモンドウォーター** | 様々な家屋 | 正気度と体力を回復 |
| **ジュース** | キッチンカウンター | 水泳のスタミナブースト |
| **フレアガン** | 初期浮遊家屋 | 霧を照らし、場所を知らせる |
| **ダイビングヘルメット** | 灯台の桟橋 | レベル8への移行に必要 |

<Callout type="warning" title="霧の危険">
霧が出現しても、パニックにならず、むやみにボートを運転しないでください。灯台の光が霧を突き抜けるまで静止するか、ゆっくり移動してください。あまりにも遠くまで進むと、光を見つけにくくなる可能性があります。
</Callout>

<Steps>
  <Step num="1" title="開始エリアを探索する">

  </Step>

  </Step>
  <Step num="3" title="フレアガンを見つける">
    近くの家屋を捜索しましょう。特定の家屋にはフレアガンとメモがあり、これらは霧が発生した際の視界確保に不可欠です。
  </Step>
  <Step num="4" title="灯台を追う">
    霧が発生し、灯台が点灯したら、シャークタンクのある家屋に到着するまで、光源に向かってまっすぐボートを進めましょう。
  </Step>
</Steps>

<Section icon={Swords} title="降下：The Thingとの遭遇">

灯台の家屋に到着すると、ゲームプレイは探索からスクリプト化されたホラーシーケンスへと移行します。シャークタンクとして知られる構造物の近くに4つのダイビングヘルメットがあります。ヘルメットを装着することは進行するために必須です。タンクに入ると、それは海の底へと降下し始めます。

この降下中に、プレイヤーは**レベル7のThe Thing**に遭遇します。このエンティティは、水中を徘徊する巨大なヘビのような生物です。スクリプトイベントで、エンティティはシャークタンクを攻撃して破壊し、あなたを外洋に放り出します。この移行は、**Escape the Backrooms レベル7**で最も激しい瞬間の1つであり、破片を避けて素早く反応する必要があります。

| フェーズ | プレイヤーの行動 | エンティティの行動 |
| :--- | :--- | :--- |
| **タンク降下** | ガラス越しに観察 | タンクを旋回し、緊張を高める |
| **破壊** | 衝突に備える | タンクを破壊する |
| **外洋** | 海底に向かって泳ぐ | 周囲を徘徊する（直接接触を避ける） |

<Callout type="success" title="サバイバル戦略">
タンクに対するThe Thingの攻撃は避けられません。ガラスが割れたら、戦おうとせず、海底の岩構造物に向かってひたすら泳ぐことに集中してください。
</Callout>

</Section>

<Section icon={Shield} title="水中ナビゲーション：洞窟の発見">

レベル7の最終セグメントでは、海底をナビゲートする必要があります。視界は非常に悪く、深海の圧力により緊急性が増します。あなたの目標は、洞窟の入り口がある特定の岩構造物を見つけることです。この洞窟がレベル8へ続く唯一の出口です。

海底には岩と砂が散乱しています。人工的な光源を探してください。洞窟の入り口は、深海の青黒い闇の中で目立つ内部の光によって示されています。洞窟に入ると、最終的に水位が下がるか、次のレベルの岩のトンネルに直接移行する道が見つかるでしょう。

**水中比較:**
| 水面レベル | 深海底 |
| :--- | :--- |
| 高い視界（霧発生前） | 極めて低い視界 |
| ボートによる移動 | 泳ぎと這いずり |
| 複数の家屋/戦利品 | 単一の出口 |
| 雰囲気的な恐怖 | 直接的なエンティティの脅威 |

<Callout type="info" title="視覚的なヒント">
洞窟を見つけるのに苦労している場合は、海底で最も明るい点を探してください。開発者は、酸素や正気度が不足しているプレイヤーの道標となるように、洞窟内にライトを配置しています。
</Callout>
