<Section icon={Map} title="レベル0概要：ロビーの探索">

「ロビー」として有名なレベル0は、Escape the Backroomsの導入体験です。繰り返される黄色の壁紙、湿ったカーペット、蛍光灯の絶え間ないハム音で、リミナルスペースの美学を完璧に捉えています。最初は空っぽに見えるかもしれませんが、このレベルは非ユークリッド幾何学とランダムなスポーンポイントによってプレイヤーを混乱させるように設計されています。

**動画のハイライト:**
- 暗い柱の部屋のようなランドマークを特定する。
- 梯子の部屋にある4つの梯子の部品をすべて見つける。
- パイプセクションでのシミーテクニックを実演する。
- 黒い矢印を使って代替ルートをナビゲートする。

| 特徴 | 説明 | 危険度 |

| **壁紙** | 単調な黄色、反復的なパターン | 低（方向感覚の喪失） |

| **音** | 一定の電気ハム音 / 遠くのノック音 | 中 |
| **エンティティ** | 上層階では稀、ピットでは一般的 | 高 |

<Callout type="tip" title="ナビゲーションの秘密">
完全に迷ったと感じたら、床に散らばった紙を探してください。それらはしばしば伝承を含んでいますが、その配置はプレイヤーが以前に探索した経路を示唆していることがあります。

<Section icon={Search} title="梯子の部屋と鍵の収集">

Escape the Backrooms レベル0の最初の主要なパズルは、壊れた梯子を修理して高所の貯蔵エリアに到達することです。この部屋は、他の黄色の廊下とは異なり、進行に不可欠です。マルチプレイヤーセッションでは、連携が重要です。チーム全体で探索できますが、最終的な梯子を操作して全員に鍵を与える必要があるのは1人だけです。

| アイテム | 目的 | 場所 |
| :--- | :--- | :--- |
| **梯子の部品 (x4)** | アクセス梯子を修理する | 梯子の部屋内に散らばっている |
| **貯蔵庫の鍵** | 出口のドアを開ける | 梯子の部屋の上棚 |
| **ロープ** | 落下したチームメイトを救助する | ピット/通気口の入口付近 |

<Steps>
  <Step num="1" title="梯子の部屋を見つける">
    柱の部屋のランドマークから、壁にもたれかかった背の高い壊れた梯子のある部屋を見つけるまで進みます。
  </Step>
  <Step num="2" title="部品を回収する">
    この部屋の角や木箱の後ろをすぐに探します。合計で4つの木製部品があります。**F**キーを押して拾います。
  </Step>
  <Step num="3" title="組み立てて登る">
    梯子の根元を操作して修理アニメーションを開始します。頂上まで登って銀色の鍵を掴みます。
  </Step>
  <Step num="4" title="道を開く">
    廊下に戻り、重い金属製のドアを探します。鍵を使ってドアを開け、ピットセクションに進みます。
  </Step>
</Steps>

<Callout type="warning" title="マルチプレイヤー同期">
鍵が拾われたら、すべてのプレイヤーが近くにいることを確認してください。鍵は「共有」されますが、一部の更新では、ドアの操作が全員に正しくトリガーされるために、プレイヤーが同じゾーンにいる必要がある場合があります。
</Callout>

</Section>

<Section icon={Shield} title="ピットセクションとバランスメカニクス">

ロックされたドアを通過した後、レベル0で最も機械的に挑戦的な部分であるピットに遭遇します。このエリアでは、暗い深淵の上に細いパイプを渡る必要があります。落下しても即死ではありませんが、エンティティでいっぱいの、はるかに危険なサブエリアに落ちることになります。

<CardGrid cols={2}>
  <Card title="バランス方式">
    - **Q**と**E**を使って体を傾けます。
    - ゆっくりと動き、揺れに注意してください。
    - 高いマウス精度を持つプレイヤーに最適です。
  </Card>
  <Card title="シミー方式">
    - 左の壁に完全に密着します。
    - 下を見ずに横向きに移動します。
    - 一般的に最も安全な「プロ」の戦略とされています。
  </Card>
</CardGrid>

チームでプレイしていて誰かが落ちた場合、**ロープ**が最も貴重なツールとなります。上に残ったプレイヤーは、左クリックボタンを押し続けることでロープを落とすことができます。ピットにいるプレイヤーは上を見てロープを見つけ、**F**キーを押して安全な場所に戻される必要があります。

| アクション | コントロール | 結果 |
| :--- | :--- | :--- |
| **左に傾く** | Q | 右側の揺れを打ち消す |
| **右に傾く** | E | 左側の揺れを打ち消す |
| **ロープを落とす** | 左クリック (長押し) | ロープをピットに降ろす |
| **しゃがむ/通気口** | Ctrl | 最終的な出口経路に入る |

<Callout type="success" title="安全な通過">
めまい効果（心臓の鼓動音）に苦しんでいる場合は、3秒間動きを止めてください。これによりキャラクターが「落ち着き」、カメラの揺れが大幅に軽減されます。
</Callout>

</Section>

<Section icon={Swords} title="下層でのサバイバル">

Escape the Backrooms レベル0でピットに落ちることは、ゲームオーバーではありませんが、「サバイバルホラー」フェーズが始まります。下層は薄暗く、レベルで唯一のアクティブなエンティティが潜んでいます。このクリーチャーは速く、音に反応するため、通気口の下にいる場合はステルスが必須です。

**ピットからの脱出方法:**
1. **矢印に従う**: 壁に描かれた黒い矢印を探してください。これらが唯一の出口へのガイドです。
2. **ノック音に耳を傾ける**: 特徴的な「ドアのノック音」は、エンティティが出現し、あなたを探していることを示します。
3. **動き続ける**: エンティティの経路はいくらか予測可能ですが、1つの部屋に長く留まっていると最終的に追い詰められます。
4. **階段**: 黒い矢印は最終的に階段へと導きます。これらの階段を上ると、出口の通気口エリアに戻ります。

| エンティティの兆候 | 意味 | 推奨される行動 |
| :--- | :--- | :--- |
| **遠くのノック音** | エンティティが出現した | 走るのをやめ、横の部屋に移動する |
| **速い足音** | エンティティが敵対した | 最も近い黒い矢印に向かって走る |
| **視覚の歪み** | エンティティが非常に近い | すぐに柱や木箱の陰に隠れる |

<Callout type="warning" title="エンティティの危険">
「ハード」または「ナイトメア」難易度では、ピット内のエンティティは著しく高速です。これらの設定では、絶対に必要でない限りピットフロアを試みないでください。
</Callout>

</Section>
