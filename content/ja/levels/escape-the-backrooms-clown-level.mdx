<Section icon={Target} title="ピエロレベルの攻略：主な目標とアイテム">
ピエロレベルでの生存は、目標を理解し、限られたリソースを効果的に利用することにかかっています。主な目標は、しばしばジェットコースターを起動することであり、それが脱出メカニズムとして機能します。このプロセスには、アニメーテッドキングを回避しながら、特定のアイテムを見つけたり、環境要素とインタラクトしたりすることが伴います。

### 生存に不可欠なアイテム

| アイテム              | 目的                                                         | 使用戦略                                                                     |
| :---------------- | :----------------------------------------------------------- | :----------------------------------------------------------------------------- |
| **懐中電灯**    | 暗闇を照らし、一時的にピエロを追い払うことができます。     | 主な手に持ち、ピエロが近づきすぎたり再出現したりしたらピエロに向けます。     |
| **ライダー（LiDAR）スキャナー** | 周囲を色付きの点でマークし、隠された経路を明らかにします。   | 懐中電灯が効果的でない複雑なエリアでのナビゲーションに不可欠です。           |
| **アーモンドウォーター**  | 正気を回復させ、精神崩壊を避けるために不可欠です。         | 正気度が低い場合はすぐに飲みます。常に少なくとも1つはインベントリに入れておきます。 |
| **ジュース**         | （アーモンドウォーターまたは同様の正気度回復アイテムのプレースホルダーである可能性が高い） | 正気度やその他の重要なステータスを回復するために消費します。できるだけ多く集めましょう。 |

<Callout type="tip" title="リソース管理">
アーモンドウォーターと懐中電灯の収集を優先してください。これらがなければ、ピエロレベルの攻略は著しく困難で危険なものになります。
</Callout>

<Section icon={Swords} title="ピエロとの遭遇戦略：ソロ vs. 協力プレイ">
アニメーテッドキングへの対処方法は、ソロプレイかチームメイトとのプレイかによって大きく異なります。どちらのシナリオでも、生存を確実にするために異なる戦術が必要です。

### ソロ生存戦術

| 戦略            | 説明                                                         | リスクレベル |
| :------------------ | :----------------------------------------------------------- | :--------- |
| **常時視認** | 常にピエロに顔を向けます。視線を維持するために後退するか、横移動します。 | 中         |
| **環境認識** | 家屋や障害物を利用して一時的に追跡を中断し、隠れます。       | 高         |
| **迅速なアイテム使用**  | 正気度が低下したら、すぐにアーモンドウォーターを使用できるように準備します。 | 中         |
| **ライダー（LiDAR）ナビゲーション** | ライダー（LiDAR）スキャナーに大きく依存して安全な経路をマッピングし、行き止まりを避けます。 | 低         |

### 協力プレイ生存戦術

| 戦略            | 説明                                                         | リスクレベル |
| :------------------ | :----------------------------------------------------------- | :--------- |
| **指定された監視者** | 一人のプレイヤーがピエロに視線を維持し、もう一人がタスクを実行します。 | 低         |
| **懐中電灯の連携** | プレイヤーは交代でピエロを照らしたり、お互いの背中をカバーしたりします。 | 中         |
| **ライダー（LiDAR）スキャナー使用者** | 専任のプレイヤーがスキャナーを使用してチームをレベル全体で誘導します。 | 低         |
| **正気度監視** | チームメンバーは正気度が低いときに声を出し、調整されたアーモンドウォーターの使用を可能にします。 | 低         |

<Section icon={Map} title="ジェットコースターの起動：脱出への道">
ピエロレベルの最終目標は、ジェットコースターを起動することです。これにより、次のステージへ、あるいはバックルームから脱出することができます。このプロセスには一連の特定の行動が必要であり、アニメーテッドキングに対処しながら慎重に実行する必要があります。

<Steps>
  <Step num="1" title="ジェットコースターを見つける">
    レベルを探索してジェットコースターのエリアを見つけます。これは通常、混沌とした環境の中で目立つランドマークです。
  </Step>
  <Step num="2" title="フィギュアパズルを解く">
    すべてのフィギュアを城エリア内の対応する台座に移動させます。このステップは、ジェットコースターへの道を開くために不可欠です。
  </Step>
  <Step num="3" title="ピエロの視認を維持する">
    ドアが開いた後も、ピエロから目を離さないでください。この重要な段階でピエロを見失わないことが非常に重要です。
  </Step>
  <Step num="4" title="正気度とアイテムを管理する">
    ジェットコースターの近くに立ちます。正気度が低い場合は、アーモンドウォーターを1つ飲みます。メインの手に懐中電灯を持ち、ポケットに少なくとも1つのアーモンドウォーターがあることを確認します。
  </Step>
  <Step num="5" title="ジェットコースターシーケンスを開始する">
    数秒間ピエロから目をそらし、その後振り返ります。ライトが消えたら、カメラを水平方向に回転させ始めます。このテクニックは、ピエロが攻撃するのを防ぐのに役立ちます。ジェットコースターの準備が整ったら、「F」を押して起動し、レベルから脱出します。
  </Step>
</Steps>
