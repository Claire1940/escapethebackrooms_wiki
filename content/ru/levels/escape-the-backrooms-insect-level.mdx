Путь через Уровень 5 часто включает поворот направо на ключевых перекрестках и следование определенным трубным образованиям. Есть определенные комнаты, которых следует полностью избегать, особенно те, у которых нет четкого выхода или которые не помечены зелеными указателями выхода. Открытие красных дверей без зеленого указателя выхода приведет вас в улей мотыльков, что смертельно опасно.

| Тип ориентира | Описание | Действие |
| :------------ | :---------- | :----- |
| **Термометр** | Отображает текущую температуру | Используйте для обнаружения мотыльков (120°F+ означает мотылька) |
| **Красные вентили** | Настенные, выпускают тепло | Нажмите, чтобы заблокировать преследующих мотыльков |
| **Трубы** | Распространены в Котельной | Следуйте по ним для правильного пути |
| **Красные двери** | Нет зеленого указателя выхода | **НЕ ОТКРЫВАТЬ** (ведет в улей мотыльков) |
| **Размытый экран** | Визуальное искажение | Мотылек рядом, выключите фонарик и присядьте |
