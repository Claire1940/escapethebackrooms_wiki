<Section icon={Target} title="Métodos alternativos para usuarios de Mac">
Dado que no existe una **descarga nativa de Escape the Backrooms para Mac**, los jugadores deben recurrir a soluciones alternativas. El método más fiable implica crear un entorno Windows en tu Mac, lo que te permite instalar y ejecutar el juego como si estuvieras en un PC con Windows.

<Steps>
  <Step num="1" title="Instalar Windows a través de Boot Camp">
    El Asistente Boot Camp es una utilidad integrada en macOS que te permite instalar Microsoft Windows en tu Mac basado en Intel y alternar entre macOS y Windows. Este es generalmente el método más recomendado y de mejor rendimiento para ejecutar juegos exclusivos de Windows.
  </Step>
  <Step num="2" title="Instalar Windows a través de software de máquina virtual">
    Software como Parallels Desktop o VMware Fusion te permite ejecutar Windows como una máquina virtual dentro de macOS. Aunque es más conveniente para alternar entre sistemas operativos, el rendimiento podría ser ligeramente inferior en comparación con Boot Camp debido al uso compartido de recursos.
  </Step>
  <Step num="3" title="Comprar e instalar el juego en Windows">
    Una vez que Windows esté instalado y funcionando correctamente en tu Mac (ya sea a través de Boot Camp o una VM), podrás comprar "Escape the Backrooms" a través de Steam e instalarlo dentro de tu entorno Windows.
  </Step>
  <Step num="4" title="Buscar servicios de juegos en la nube (éxito limitado)">
    Algunos jugadores exploran servicios de juegos en la nube que transmiten juegos desde potentes servidores remotos. Sin embargo, muchos de estos servicios, incluidos algunos populares, no ofrecen actualmente "Escape the Backrooms" en sus bibliotecas. Incluso si puedes cargar tu biblioteca de Steam, es posible que el juego no funcione.
  </Step>
</Steps>

<Callout type="tip" title="Boot Camp vs. Máquina virtual">
Para el mejor rendimiento de juego, **Boot Camp es generalmente preferible** ya que permite a Windows acceder directamente al hardware de tu Mac. Las máquinas virtuales, aunque flexibles, introducen una capa de abstracción que puede afectar la velocidad de fotogramas y la capacidad de respuesta general.
</Callout>

<Section icon={Shield} title="Requisitos del sistema y consideraciones de rendimiento">
Comprender los requisitos del sistema para "Escape the Backrooms" es crucial al intentar ejecutarlo en un Mac, especialmente a través de métodos no nativos. El juego, disponible en Steam, enumera necesidades de hardware específicas para Windows que impactan directamente tu experiencia en macOS.

| Componente | Requisito Mínimo (Windows) | Requisito Recomendado (Windows) | Consideraciones para Mac (vía Boot Camp) |
|---|---|---|---|
| **SO** | Windows 10/11 64 bits | Windows 11 64 bits | Requiere Boot Camp para Mac Intel; Parallels/VMware para Mac M1/M2 (Windows ARM) |
| **Procesador** | Intel o AMD de cuatro núcleos 2.5 GHz | Intel Core i5-10400 / AMD Ryzen 5 3600 | El rendimiento depende de la CPU del Mac, generalmente bueno para Mac Intel |
| **Memoria** | 8 GB RAM | 16 GB RAM | Igualar o superar para una jugabilidad fluida |
| **Gráficos** | Cualquier tarjeta compatible con DirectX 11 o 12 | NVIDIA GeForce RTX 2060 / AMD Radeon RX 6600 | Los gráficos integrados (ej., Intel Iris Xe) pueden tener dificultades; una GPU dedicada (ej., AMD Radeon) es mejor |
| **DirectX** | Versión 11 | Versión 11 | Gestionado por la instalación de Windows |
| **Almacenamiento** | 25 GB de espacio disponible | 25 GB de espacio disponible | Asigna suficiente espacio para la partición de Windows |

<Callout type="info" title="Usuarios de Mac M1/M2">
Para los Mac con Apple Silicon (M1, M2, etc.), Boot Camp no está disponible. Tu única opción para ejecutar Windows es a través de software de virtualización como Parallels, que es compatible con las versiones ARM de Windows. El rendimiento puede variar y la compatibilidad con todos los juegos de Windows no está garantizada.
</Callout>
